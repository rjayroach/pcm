## Overview

Update all plugin credential definitions to use `provider:` instead of `vault:`.
This is a simple find-and-replace across the pcm-plugins repo.

## Implementation

### 1. gh plugin

Update `~/spaces/rjayroach/technical/pcm-plugins/plugins/gh/credentials/gh.yml`:

```yaml
# GitHub CLI credentials

credentials:
  gh:
    provider: user
    fields:
      token:
        env: GH_TOKEN
```

Change: `vault: personal` → `provider: user`

### 2. aws plugin

Update `~/spaces/rjayroach/technical/pcm-plugins/plugins/aws/credentials/aws.yml`:

```yaml

    provider: workspace

      access-key-id:
```
